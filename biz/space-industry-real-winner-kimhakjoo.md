## 한 줄 요약

우주산업의 진짜 승자는 SpaceX 같은 발사체 기업이 아니라, 위성 간 레이저 통신·광학 부품·방열 솔루션·특수 소재·배터리 등 "보이지 않는 핵심 부품 기업"들이다. 특히 **L3 해리스 테크놀로지**가 위성의 꽃이라 할 수 있다.

---

## 핵심 메시지

1. **AI·로봇·위성은 세트** — 이 셋이 결합되어 새로운 산업 인프라를 형성하며, 위성은 통신 백본 역할
2. **저궤도 위성이 통신 패러다임을 바꾼다** — 고궤도(35,786km) → 저궤도(300~800km)로 거리 100분의 1 축소
3. **발사체(SpaceX)는 "배달원"** — 진짜 가치는 위성 탑재물과 부품에 집중
4. **위성 간 레이저 통신이 핵심 기술** — 우주 공간에서 빛의 속도로 직통 통신, 지상 광케이블 우회
5. **인플레이션 해결책** — 로봇 공장을 경오지(외딴 곳)에 배치 → 땅값·인건비·에너지 비용 하락 → 물가 안정

## 챕터별 상세 정리

### 1. 왜 위성이 중요한가 — AI·로봇·위성의 결합 (00:00 ~ 04:40)

**통신 세대의 발전:**
- 4G (LTE): 0.7 ~ 2.6 GHz
- 5G: 3.5 GHz ~ 28 GHz (선폭이 좁을수록 더 많은 데이터 전송 가능)
- 6G: 위성이 중심이 될 것

**AI·로봇 시대의 위성 역할:**
- 피지컬 AI, 로봇 등장 → 로봇 간 통신 필요 → 위성이 답
- 로봇은 **실시간 + 빈번한 통신** 필요 → 위성이 최적
- 다크 팩토리(무인 공장)는 인간 출퇴근 불필요 → 경오지에 배치 가능
- 경오지엔 유무선망 없음 → 위성 통신 필수

**인플레이션 완화 메커니즘:**
- 로봇 공장을 오지에 배치 → 땅값 하락
- 로봇이 인간보다 생산성 높음 → 인건비 압력 감소
- 소형 원자로(SMR, MMR, USAR)로 에너지 비용 절감
- 땅값 + 인건비 + 에너지 비용 ↓ = 인플레이션 완화

### 2. 군사적 가치 — 신냉전 시대의 위성 (04:40 ~ 07:00)

- 1950년대 냉전에서 위성 경쟁 시작 (정찰 목적)
- 현대: 미국·중국·러시아 모두 군사 위성 운용
- SpaceX가 1만 개 이상 위성 발사 → 군사용 전환 가능
- SMR(1MW급, 1톤)을 위성에 탑재 가능 → ICBM 디코이(가짜 목표물) 생성, 유도 신호 교란
- 군사적 인센티브만으로도 위성 투자 충분

### 3. 위성 통신의 기술적 한계와 해결책 (07:00 ~ 12:00)

**위성 간 통신 = 레이저:**
- 우주는 진공 → 빛의 속도 100% 활용 (광케이블은 유리 저항으로 ~2/3 속도)
- 위성끼리 레이저로 직통 → 대륙간 통신도 우주에서 릴레이가 더 빠름

**지상↔위성 통신 = 전자기파:**
- 레이저는 구름/비/불순물에 튕겨나감 → 전자기파 사용
- 전자기파는 선폭이 길어 데이터 용량 제한
- 고용량 데이터 전송은 어렵지만, 경량 센서 데이터에 최적

**저궤도 위성의 장점과 과제:**
- 고궤도: 35,786km vs 저궤도: 300~800km → 거리 100배 축소
- 문제: 중력 영향으로 궤도 이탈 → 빨리 달려야(초속 7.5km) 원심력 유지
- 빨리 달리면 지상 시야에서 빨리 사라짐 → **위성이 많아야** 연속 커버리지 확보
- SpaceX 혁신: 위성 부품 표준화 + 컨베이어 벨트 생산 + 엔진 재활용 → 비용 혁신

**빔포밍(Beamforming):**
- 지상↔위성 전자기파 통신 강화 기술
- 안테나 소자를 고밀도로 배치 → 주기(위상) 맞추면 보강 간섭 → 에너지 집중
- 레이저 흉내를 내는 전자기파 압축 기술
- 이 기술 잘하는 곳: **AST 스페이스 모바일** (상장 기업)
- D2C(Direct to Cell): 위성에서 휴대폰으로 직접 전송 가능

### 4. 위성 통신의 실제 응용 분야 (12:00 ~ 20:00)

**적용 분야별 위성 적합도:**

| 분야 | 데이터 유형 | 위성 적합도 |
|------|-----------|-----------|
| 소형 원자로 제어 | 온도 센서 (경량) | ⭐⭐⭐ 최적 |
| 신재생 에너지 발전 | 전기 데이터 (경량) | ⭐⭐⭐ 최적 |
| 바이오/질소 농도 제어 | 센서 데이터 (경량) | ⭐⭐⭐ 최적 |
| 반도체 **후공정** | 전압/전류/응답속도 (중량) | ⭐⭐ 적합 |
| 반도체 **전공정** | 나노미터 패턴 이미지 (고용량) | ⭐ 부적합 |
| 재련(배터리→니켈코발트망간) | 화학 반응 센서 (경량) | ⭐⭐⭐ 최적 |
| 화학/정유 공장 | 반응 속도 데이터 (경량) | ⭐⭐⭐ 최적 |

**내비게이션/자율주행:**
- GPS는 고궤도(20,000km) → 신호 약함, 오차 큼
- 저궤도 위성으로 보완 → 오차 1m 이내, 어디서나 잡힘(Ubiquitous)
- 터널, 빌딩 숲, 계곡, 비행기 안에서도 연결 → 통신사 차별화 전략
- 자율주행 트럭 → 운전자 인건비 절감 → 위성 내비 필수

**위성 정착 시기:** 교수 예상 5~6년 후, 다만 통신사 차별화 수요로 빨라질 수 있음

### 5. SpaceX 밸류에이션 (20:00 ~ 21:00)

- 연 영업이익 ~80억 달러, 시가총액 ~1조 달러 → PER ~125배
- 겉보기 비싸지만: 경오지 비즈니스 확대 + 장거리 경량 통신 우위 감안하면 "시작점"
- **그러나 진짜 기회는 부품 업체에 있다**

**AI 반도체로의 확장:** 구리선 인터포저 → 광 인터포저 전환 시 코닝의 유리 기판 수요 폭발. 삼성도 삼성디스플레이(유리 패턴 경험) + 삼성전기(유리 기판 역량)로 따라잡기 가능하지만, TSMC가 이미 앞서있음.

**방열의 중요성:**

- 냉매: 암모니아(기화점 -33°C)로 열 흡수, 액체 나트륨(끓는점 883°C)로 태양 열 전달

## 💡 교수님이 가장 강조한 기업

### **L3 해리스 테크놀로지 (LHX)**
> "위성의 꽃"

- 위성 통신 탑재 전체 솔루션: 송수신기 + 증폭기 + 방열 솔루션
- 에어로젯 로켓다인(로켓 전체 솔루션)도 인수
- 자동차로 치면 **현대모비스** 같은 존재 — 겉엔 안 보이지만 뒤에서 핵심 부품 전부
- 현재 주가는 방산주 전반 하락으로 눌림 → 미국 전쟁 타협 시 매수 기회
